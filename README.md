**TODO List App API**

**Overview:** <br>
Language: `Go` <br>
DB: `Mongo` <br>


**On Action:** <br>
`git clone https://github.com/hbrawnak/todo-list-app.git` <br>
`cd todo-list-app` <br>
`go get go.mongodb.org/mongo-driver/mongo` <br>
`go get -u github.com/gorilla/mux`

Setup db config in `settings/db.go`  <br>
And `go run main.go`

BOOM!!
